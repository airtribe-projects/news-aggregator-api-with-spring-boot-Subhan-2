# 📰 News Aggregator API

This is a Spring Boot-based RESTful API that provides user authentication using JWT and personalized news aggregation using external APIs. Users can register, log in, manage preferences, and retrieve news based on their selected topics.

---

## 📌 Features

- ✅ User Registration with preferences
- ✅ Secure Login with JWT authentication
- ✅ Password encryption using BCrypt
- ✅ Stateless session management (JWT)
- ✅ Preferences management (GET/PUT)
- ✅ News fetching from external APIs (e.g., NewsAPI.org)
- ✅ Spring Boot + Spring Security + H2 database

---

## 🛠 Tech Stack

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- H2 (In-Memory DB)
- Maven
- Postman for testing
